# Feedback Collection System


## Getting Started
To run the React frontend and the server, follow these steps:

### Terminal 1 (Frontend)
1. Navigate to the frontend directory:

   ```bash
   cd frontend

2. Install dependencies:

   ```bash
   npm install

3. Run the React app:

   ```bash
   npm run dev

### Terminal 2 (Server)

1. Navigate to the server directory:

   ```bash
   cd server

2. Install dependencies:

   ```bash
   npm install

3. Run the React app:

   ```bash
   npm start
